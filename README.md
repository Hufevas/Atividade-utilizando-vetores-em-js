A aplicação permite registrar vendas com nome do vendedor e valor, aplicando automaticamente um desconto de 10%, além de gerar ID sequencial e armazenar data e hora do cadastro.

Inclui funcionalidades de adição, remoção individual, remoção do último item e limpeza completa da lista, utilizando manipulação de DOM e métodos de array.

Projeto desenvolvido para fins educacionais na disciplina de Programação Web.
